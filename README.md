# Sistem Penyiraman Otomatis dengan NodeMCU ESP8266 dan Sensor Kelembapan

Source code ini adalah projek dari mata kuliah masyarakat cerdas.
Sistem yang dibangun menggunakan beberapa komponen berikut:
- NodeMCU ESP8266
- Sensor Kelembapan
- LCD I2C 16x2
- Relay 5 Volt
- Pompa Air
- Sprayer
- Solar Panel
